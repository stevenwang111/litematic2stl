# Litematica to STL

Convert Minecraft Litematica schematics to 3D-printable STL files with a live 3D preview.

## Features

- **Convert** `.litematic` files to binary `.stl` format
- **Preview** STL models in an interactive 3D viewer (rotate / zoom / pan)
- **Download** the converted STL directly
- **Color theming** – dark & light mode for the viewer
- **Lighting modes** – switch between point light and ambient light
- **Model info** – face count, vertex count, bounding box size, and estimated volume
- **Drag-and-drop** support for `.stl` files directly into the viewer
- **Open existing STL files** – load and inspect any `.stl` file

## How it works

The project consists of two parts:

- **Backend** (`server.py`) – A Flask server that reads Litematica schematic files using `litemapy`, converts recognized blocks into triangular mesh geometry, and outputs a binary STL file.
- **Frontend** (`static/`) – A Three.js-based 3D viewer that renders the STL in the browser with full lighting, shadow, and orbit controls.

### Supported Blocks

| Block               | Geometry                          |
|---------------------|-----------------------------------|
| Any colored wool    | Full cube (1×1×1)                 |
| Oak slab            | Half slab (bottom / top / double) |
| Oak stairs          | Stair geometry with facing, half, and corner shape (straight / inner / outer) |

### Wool Color Map

All 16 Minecraft wool colors are mapped to their in-game RGB values, allowing the backend to preserve color information (the STL file itself stores geometry only; colors are used for display in the game).

## Requirements

- Python 3.8+
- Flask
- litemapy

## Installation

```bash
pip install flask litemapy
```

## Usage

```bash
python server.py
```

Open your browser and navigate to `http://localhost:8080`.

- Click **Upload Litematic** to select a `.litematic` file – it will be converted to STL and displayed in the viewer.
- Click **Open STL** to load an existing `.stl` file for preview.
- Click **Download STL** to save the converted file.
- Drag and drop `.stl` files directly onto the canvas.

### API Endpoint

**POST** `/convert` – Upload a `.litematic` file and receive a binary `.stl` response.

- Request: `multipart/form-data` with field name `file`
- Response: `application/octet-stream` (binary STL)

## Motivation
I’ve always been fascinated by 3D modeling and 3D printing, but professional tools like Blender come with a steep learning curve that can be intimidating. Minecraft, on the other hand, is a game I know well—its block-based building is intuitive, simple, and enjoyable. So I thought: why not use Minecraft as my “modeling software” and then convert those builds into the standard STL format for 3D printing? This tool was born exactly from that idea—it lets me bypass complex modeling software and bring my ideas to life in a way I’m already comfortable with.(Actually, I don't have a 3D printer. If there are any issues with printing the generated model, please adjust it yourself or submit an issue)
